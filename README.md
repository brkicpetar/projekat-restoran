## About project
Project "**Restaurant Application**" is intended for use in restaurants. This software has **CLI environment** and is intended for use on **Windows** machines.

### Main features
* food and beverages menu
* bill generation, VAT and discount calculations
* table reservations
* phone ordering
* guestbook
* management of restaurant (food and beverages menu changes, guestbook cleaning)
