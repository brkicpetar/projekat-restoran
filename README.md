# Projekat "Aplikacija za Restoran"/Project "Restaurant Application"

### Srpski

## O projektu
Projekat "**Aplikacija za restoran**" je namenjen da pomogne u radu restorana i sličnih ugostiteljskih objekata. Interfejs ovog programa je **konzolno-linijski** i namenjen je računarima sa operativnim sistemom **Windows**.

### Glavne funkcionalnosti
* meni jela i pića
* generisanje računa, obračun PDV-a i raznih popusta
* rezervacija stolova
* naručivanje preko telefona
* knjiga utisaka
* izmena jelovnika, karte pića, brisanje podataka iz knjige utisaka...

### English

## About project
Project "**Restaurant Application**" is intended for use in restaurants and cafés. This software has **CLI environment** and is intended for use on **Windows** machines.

### Main features
* food and beverages menu
* bill generation, VAT and discount calculations
* table reservations
* phone ordering
* guestbook
* management of restaurant (food and beverages menu changes, guestbook cleaning)
